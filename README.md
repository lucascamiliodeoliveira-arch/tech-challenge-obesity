# Tech Challenge - Predição de Obesidade

## Objetivo

Desenvolver um modelo de Machine Learning capaz de prever o nível de obesidade de indivíduos com base em características físicas, hábitos alimentares e estilo de vida.

## Dataset

Base de dados de obesidade contendo informações sobre:

* Idade
* Altura
* Peso
* Consumo de água
* Atividade física
* Histórico familiar
* Consumo de álcool
* Meio de transporte
* Hábitos alimentares

## Pipeline de Machine Learning

1. Análise exploratória dos dados
2. Tratamento e preparação dos dados
3. Feature Engineering
4. Treinamento do modelo Random Forest
5. Avaliação do desempenho
6. Deploy em Streamlit

## Modelo Utilizado

Random Forest Classifier

## Resultado

Accuracy: 95,74%

## Principais Variáveis

* Weight
* Height
* Age
* FCVC
* Gender

## Aplicação

Sistema preditivo desenvolvido em Streamlit para auxiliar a identificação dos níveis de obesidade.
